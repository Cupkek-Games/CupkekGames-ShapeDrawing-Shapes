# CupkekGames ShapeDrawing Shapes

Concrete `Indicator` implementations for `com.cupkekgames.shapedrawing`, drawn with the Freya Shapes addon.

## What's inside

**Runtime** (`CupkekGames.ShapeDrawing.Shapes.asmdef`)

- `IndicatorCircle` — circle indicator
- `IndicatorLine` — line indicator
- `IndicatorArc` — arc indicator

## Dependencies

- `com.cupkekgames.shapedrawing` (asmdef ref)
- Freya Shapes addon (asmdef ref, bring your own copy)
